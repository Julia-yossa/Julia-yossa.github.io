# Julia Yossa - Personal & Professional Portfolio

Welcome to the source code for my personal portfolio website. This project is a responsive, single-page application designed to showcase my skills, projects, and professional background as an IT Professional. It is built with modern web technologies and features a clean, dark-themed, and minimalist design.

**[➡️ View Live Demo](https://Julia-yossa.github.io/)**

---

## ✨ Key Features

*   **Fully Responsive Design:** Looks great on mobile, tablet, and desktop screens.
*   **Bilingual Content:** Easily toggle between English and French content across the entire site.
*   **Dynamic Typing Animation:** An engaging typing animation cycles through my various professional titles.
*   **Comprehensive Skills Section:** A detailed showcase of 10 core competency areas, from Network Administration to Business Analysis.
*   **Interactive Project Cards:** Each project includes a description, technologies used, and links to a live demo and the source code.
*   **Semantic & Accessible:** Built with semantic HTML5 and ARIA attributes for improved accessibility.
*   **Pure Vanilla JavaScript:** All interactivity is handled with clean, dependency-free JavaScript.

---

## 🛠️ Tech Stack

This project was built from scratch using:

*   **HTML5:** For the core structure and content.
*   **Tailwind CSS:** For a utility-first, responsive, and modern design.
*   **Vanilla JavaScript:** For all interactivity, including the language toggle, mobile menu, and typing animation.
*   **Font Awesome:** For high-quality icons in the skills section.
